Guía: Cómo instalar OneCore API en Windows XP

OneCore API es un conjunto de archivos de sistema que le permiten a Windows XP ejecutar versiones modernas de aplicaciones que oficialmente dejaron de darle soporte a dicho sistema operativo.

El proceso para instalar OneCore API puede ser algo enredado a comparación del proceso de Extended Kernel para Windows Vista, por lo que deberán ser pacientes y se recomienda el uso de máquina virtual en caso de que se produzcan problemas con el arranque del sistema operativo (Nota: a partir de la versión 3.0, OneCore API debería funcionar de manera más estable en hardware real, pero igual se recomienda que usen una máquina virtual primero para probarlo). Dicho esto, vamos con la instalación.

Importante: Los archivos requeridos se pueden descargar aquí (Nota: si les aparece que el archivo .zip tiene virus o algo así, agreguen el archivo como excepción, porque es un falso positivo y no lo vamos a instalar en un sistema operativo que no sea Windows XP de todos modos).

Prerrequisitos: Windows XP SP3 o Windows XP x64 SP2 instalado, idealmente en inglés y actualizados (pueden instalar las actualizaciones con Legacy Update).

Paso 1: Descarga el archivo ZIP del Github vinculado arriba. Luego, extrae la carpeta x86 o x64 usando WinRAR o 7-Zip, dependiendo de la arquitectura de su procesador, la cual pueden revisar haciendo clic derecho en Mi PC o My Computer y clic en Propiedades (si les aparece x64 Edition deben usar los archivos de la carpeta x64 y sino, deben usar los archivos de la carpeta x86).

Paso 2: Ejecuta los archivos de instalación en este orden: one-core-api-base.exe - one-core-api-additional-dll.exe - one-core-api-branding.exe - one-core-api-d3d.exe - one-core-api-driver-update.exe - one-core-api-app-compat.exe. Nota: la sección "Installation order of packages" del Github hace mención de archivos que no están presentes en el archivo .zip de la versión actual (3.0.5 al momento de escribir esto), por lo que simplemente puedes ignorarlos. Haz clic en "Do not restart now" al finalizar la instalación de cada archivo (si no aparece esta opción, simplemente dale a Finalizar) a excepción del último. Si aparece una alerta de Windows File Protection pidiendo que insertes el disco de instalación de Windows, simplemente dale en Cancelar.

Paso 3: Reinicia el sistema operativo. Si todo salió bien, deberías poder ejecutar aplicaciones modernas en Windows XP. Si no, intenta reinstalar OneCore API siguiendo los pasos anteriores.

¡Y listo! Para instalar aplicaciones incompatibles deben ir a Propiedades, luego Modo de Compatibilidad y elegir Windows 7 o alguna versión posterior. Algunas aplicaciones requieren de pasos extra. Por ejemplo, para Chrome u otros navegadores basados en Chromium, deben colocar un Acceso directo en el Escritorio, ir a Propiedades y pegar el siguiente texto en la ruta de la aplicación: --no-sandbox --force-wave-audio --disable-exclusive-audio.

Lista de compatibilidad de OneCore API:

  • Chromium 122 o anteriores
  • Firefox 115 o anteriores
  • SumatraPDF (última versión)
  • Notepad++ (última versión)
  • Juegos de Windows 7
  • WinRAR (última versión)
  • Zoom (última versión)
